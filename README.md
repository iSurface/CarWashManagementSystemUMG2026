# CarWashManagementSystemUMG2026
Sistema de gestión para Simulacion Car Wash desarrollado como proyecto universitario para la clase Arquitectura de Computadoras I Universidad Mariano Galvez Antigua. Permite administrar servicios, clientes y procesos de lavado. Implementado con Node.js, Arduino, HTML, CSS y JavaScript para integrar software y hardware en una solución funcional.
